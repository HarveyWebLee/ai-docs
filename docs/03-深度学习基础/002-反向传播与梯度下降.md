# 002 · 反向传播与梯度下降

> 本文回答：网络的权重是怎么被"学"出来的？反向传播和梯度下降各自扮演什么角色？

## 一、直觉与通俗解读

训练神经网络就像**蒙着眼睛下山**，目标是走到山谷最低处（损失最小）：

- **损失函数**是"海拔高度"，越低越好。
- **梯度**是"当前脚下最陡的上坡方向"，那么它的反方向就是"最快下坡方向"。
- **梯度下降**就是"每一步都朝最陡下坡方向迈一小步"。
- **反向传播**则是"如何高效算出每个参数对海拔的影响"——它利用链式法则，从山顶（输出误差）一路回溯到每个参数，避免了逐个参数暴力试探。

一句话：**反向传播负责"算梯度"，梯度下降负责"用梯度更新参数"。**

## 二、严谨定义与原理

### 2.1 损失函数

以回归的均方误差为例，单样本损失：

$$
L = \frac{1}{2}(\hat{y} - y)^2
$$

训练目标是最小化所有样本损失的平均值 $J(\theta) = \frac{1}{m}\sum_{i=1}^{m} L^{(i)}$，其中 $\theta$ 为全部权重与偏置。

### 2.2 梯度下降

参数按负梯度方向迭代更新：

$$
\theta \leftarrow \theta - \eta \, \nabla_\theta J(\theta)
$$

其中 $\eta$ 是**学习率**。学习率过大易震荡/发散，过小则收敛缓慢。

- **批量梯度下降（BGD）**：用全部样本算一次梯度，稳定但慢。
- **随机梯度下降（SGD）**：每次用一个样本，快但噪声大。
- **小批量（Mini-batch）**：折中，工业界主流。

### 2.3 反向传播（链式法则）

设某层 $z = \mathbf{w}^\top \mathbf{a} + b$，$out = \sigma(z)$，损失为 $L$。反向传播反复应用链式法则：

$$
\frac{\partial L}{\partial w_i}
= \frac{\partial L}{\partial out}\cdot\frac{\partial out}{\partial z}\cdot\frac{\partial z}{\partial w_i}
= \underbrace{\frac{\partial L}{\partial out}\,\sigma'(z)}_{\text{记为误差项 }\delta}\cdot a_i
$$

关键在于：**上一层的误差项 $\delta$ 可以由下一层的 $\delta$ 递推得到**，因此只需从输出层向输入层"回传"一次，即可得到所有参数的梯度，复杂度与前向传播同阶。这正是反向传播高效的根本原因。

## 三、案例解析：手算一次参数更新

考虑最简单的单神经元（无激活，恒等输出）：$\hat{y} = w x + b$，损失 $L=\frac{1}{2}(\hat{y}-y)^2$。

给定样本 $x=2,\ y=4$，初始 $w=0,\ b=0$，学习率 $\eta=0.1$。

**前向：** $\hat{y} = 0\cdot 2 + 0 = 0$，$L = \frac{1}{2}(0-4)^2 = 8$。

**反向（求梯度）：**

$$
\frac{\partial L}{\partial \hat{y}} = \hat{y}-y = -4,\quad
\frac{\partial L}{\partial w} = (\hat{y}-y)\,x = -8,\quad
\frac{\partial L}{\partial b} = (\hat{y}-y) = -4
$$

**更新：**

$$
w \leftarrow 0 - 0.1\times(-8) = 0.8,\qquad
b \leftarrow 0 - 0.1\times(-4) = 0.4
$$

**验证是否变好：** 新预测 $\hat{y}=0.8\times2+0.4 = 2.0$，新损失 $L=\frac{1}{2}(2.0-4)^2 = 2.0 < 8$。损失确实下降，说明一步更新有效。多次迭代后 $\hat{y}$ 将逼近 $4$。可运行版本见 `code/03-深度学习基础/`（后续补充）。

## 四、常见误区与边界

- **误区："反向传播是一种训练算法"**：不是。反向传播只是**高效计算梯度**的方法；真正更新参数的是梯度下降（及其变体 Adam、Momentum 等）。
- **误区："学习率随便设"**：学习率是最关键的超参数之一，通常需要网格/退火调度。
- **梯度消失/爆炸**：深层网络中梯度连乘可能趋于 0 或无穷，需用 ReLU、残差连接、批归一化、梯度裁剪等缓解。

## 五、小结与延伸阅读

- 损失函数定义"好坏"，梯度指出"下降方向"，梯度下降据此更新参数。
- 反向传播用链式法则从输出回溯，一次回传得到所有梯度，是训练可行的关键。
- 手算案例展示了"前向 → 求梯度 → 更新 → 损失下降"的完整闭环。
