# 000 · 分类总览与知识图谱

> 本页是「深度学习基础」分类的导读，串联本分类知识点，并用知识图谱呈现它们之间的关系。

## 一、本分类学什么

深度学习基础聚焦"**用多层神经网络从数据中自动学习特征**"这一核心思想。掌握本分类后，你应能回答：

- 神经网络由什么组成、为什么"深"比"浅"更强？（见 [001 · 神经网络结构](./001-神经网络结构.md)）
- 网络是如何"学会"的？参数怎么被自动调整？（见 [002 · 反向传播与梯度下降](./002-反向传播与梯度下降.md)）

## 二、通俗理解本分类

可以把神经网络想象成一条**流水线上的多级质检+加工工序**：原始零件（输入数据）经过一道道工序（网络层），每道工序都做一点变换，最终产出成品（预测结果）。而"训练"就是根据成品的合格程度（损失），**反向回溯**告诉每道工序"你该怎么微调"，这就是反向传播 + 梯度下降。

## 三、知识图谱

```mermaid
graph TD
  Input["输入数据 (特征向量)"] --> Neuron["神经元: 加权求和 + 偏置"]
  Neuron --> Act["激活函数 (引入非线性)"]
  Act --> Layer["网络层 (多个神经元)"]
  Layer --> Deep["多层堆叠 = 深度网络"]
  Deep --> Output["输出 (预测)"]
  Output --> Loss["损失函数 (衡量误差)"]
  Loss --> Backprop["反向传播 (链式法则求梯度)"]
  Backprop --> GD["梯度下降 (按梯度更新参数)"]
  GD --> Neuron

  classDef fwd fill:#e8f5e9,stroke:#388e3c,color:#1b5e20;
  classDef learn fill:#fff3e0,stroke:#f57c00,color:#e65100;
  class Input,Neuron,Act,Layer,Deep,Output fwd;
  class Loss,Backprop,GD learn;
```

图中**绿色**为"前向传播（推理）"路径，**橙色**为"训练（学习）"路径。二者构成一个闭环：前向算出预测与误差，反向据误差更新参数，循环往复直至收敛。

## 四、学习建议

1. 先读 [001 · 神经网络结构](./001-神经网络结构.md)，建立对"网络由什么组成"的直觉。
2. 再读 [002 · 反向传播与梯度下降](./002-反向传播与梯度下降.md)，理解"网络如何学习"。
3. 配合 `code/03-深度学习基础/` 下的可运行示例动手实验。

## 五、小结

- 深度学习 = 多层非线性变换 + 基于梯度的自动参数优化。
- 前向传播负责"预测"，反向传播 + 梯度下降负责"学习"，二者闭环迭代。
- 本分类是理解 NLP、CV、LLM 等后续分类的必备基础。
