# 003 · 微积分与梯度

> 本文用大白话回答：导数、梯度到底在说什么？为什么"朝着负梯度方向走"就能让损失越来越小？链式法则又是怎么撑起反向传播的？
>
> 读完你会知道：训练模型说白了就是"蒙眼下山找最低点"，而导数/梯度就是你脚下探到的"坡朝哪边、有多陡"。

## 一、一句话先说清

**导数告诉你"往前挪一小步、结果会怎么变"；梯度是它的多维版本，指向"上升最快"的方向，反过来就是"下降最快"的方向。**

- **导数**：某一点的**斜率**，即"瞬时变化率"。
- **梯度**：一个**向量**，指向函数上升最快的方向；负梯度就指向下降最快的方向。
- **链式法则**：一条"逐层传递影响"的规则——反向传播就是它的层层套用。

## 二、打个比方：蒙眼下山

你被蒙上眼睛，站在一座山上，想尽快走到山谷最低点。你看不见全局，只能：

1. 用脚感受**当前脚下哪个方向最陡**（这就是梯度）；
2. 朝最陡的**下坡方向**迈一步（这就是负梯度方向）；
3. 重复，直到脚下变平（走到最低点附近）。

| 下山的动作 | 数学里的对应 | 含义 |
| --- | --- | --- |
| 感受脚下坡度 | 求导数/梯度 | 当前位置往各方向走，函数怎么变 |
| 朝最陡下坡走 | 沿 $-\nabla f$ 方向 | 让函数值下降最快 |
| 每步迈多大 | 学习率 $\eta$ | 步子太大会跨过谷底，太小走得慢 |

训练神经网络，就是在"损失这座山"上不停重复这套动作。

## 三、它到底解决什么问题

### 问题 1：怎么知道"参数往哪调，损失会变小"

**朴素做法**：把参数随便加一点、减一点，各试一遍看损失变没变——参数一多就试不过来了。

**更好做法**：用**导数/梯度**一次性算出"每个参数往哪个方向、调多少最有效"。

| 做法 | 怎么找下降方向 | 代价 |
| --- | --- | --- |
| 到处试探 | 每个参数正负都试 | 参数多时爆炸 |
| 求梯度 | 一次计算得到所有方向 | 高效、有方向感 |

### 问题 2：函数是"一层套一层"的，梯度怎么算

神经网络是"输入 → 第一层 → 第二层 → … → 损失"层层嵌套的复合函数。**链式法则**解决的正是"嵌套函数怎么求导"：把整体影响拆成一段段影响相乘。

- 大白话：如果 A 影响 B、B 影响 C，那么 **A 对 C 的影响 = (A 对 B 的影响) × (B 对 C 的影响)**。
- 反向传播就是从损失往回，一段段把这些"影响"乘起来，一次回传就拿到所有参数的梯度。

## 四、专业视角（与大白话对齐）

### 4.1 导数与偏导数

单变量导数：

$$
f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}
$$

> 对齐：这就是前面说的"往前挪一小步 $h$，函数值变化 $f(x+h)-f(x)$ 除以步长"，步长趋近于 0 时得到的就是该点的斜率。

多变量函数 $f(x_1,\dots,x_n)$ 对某一个变量的**偏导数** $\dfrac{\partial f}{\partial x_i}$，就是把其它变量当常数、只看这一个变量的导数。

### 4.2 梯度

梯度是所有偏导数组成的向量：

$$
\nabla f = \left(\frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n}\right)^\top
$$

**核心性质**：$\nabla f$ 指向 $f$ 增长最快的方向，$-\nabla f$ 指向下降最快的方向。这就是梯度下降 $\theta \leftarrow \theta - \eta\nabla f$ 的数学依据。

> 对齐：这条更新式就是前面"蒙眼下山"的公式版——$-\nabla f$ 是"最陡下坡方向"，$\eta$（学习率）是"这一步迈多大"。

### 4.3 链式法则

若 $y=f(u)$ 且 $u=g(x)$，则

$$
\frac{dy}{dx} = \frac{dy}{du}\cdot\frac{du}{dx}
$$

多变量情形推广为雅可比矩阵（Jacobian）相乘。

> 对齐：这两项相乘，正是前面那句"A 对 C 的影响 = (A 对 B) × (B 对 C)"。**反向传播 = 从输出到输入反复套用链式法则，一次回传就得到所有参数的梯度。**

### 4.4 二阶信息（了解）

二阶导数（多维为 **Hessian 矩阵**）刻画的是**曲率**：在最优点附近，函数到底是"陡峭的碗"还是"平缓的盘"。曲率会影响优化难易和学习率的选择。

## 五、案例解析：用梯度找 $f(x)=x^2$ 的最小值

目标：最小化 $f(x)=x^2$，其导数 $f'(x)=2x$。取初值 $x_0=4$，学习率 $\eta=0.1$，迭代规则 $x \leftarrow x - \eta\cdot 2x = x(1-0.2)=0.8x$：

| 步 | $x$ | $f(x)=x^2$ |
| --- | --- | --- |
| 0 | 4.000 | 16.000 |
| 1 | 3.200 | 10.240 |
| 2 | 2.560 | 6.554 |
| 3 | 2.048 | 4.194 |
| ... | ... | ... |
| 10 | 0.429 | 0.184 |

$x$ 稳步趋向最优点 $0$，$f(x)$ 单调下降。这就是"沿负梯度方向反复迭代"的完整闭环——训练神经网络的最小化过程，本质上就是它的高维放大版。

## 六、常见误区与边界

- **误区："梯度为 0 就是最小值"**：梯度为 0 只说明是**驻点**，可能是极小、极大或**鞍点**；深度学习的高维损失面上到处是鞍点。
- **误区："学习率越大收敛越快"**：太大会跨过最优点甚至发散（上例若 $\eta=1.1$，$x$ 反而会越来越大）。
- **不可导点**：像 ReLU 在 0 处不可导，工程上用**次梯度**（这里取 0）来处理。

## 七、一句话总结

- 导数是斜率、梯度是"最快上升方向"，负梯度用来做最小化（下山）。
- **链式法则**把复合函数的梯度拆成逐层相乘，是反向传播的数学基础。
