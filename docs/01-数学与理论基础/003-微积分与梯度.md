# 003 · 微积分与梯度

> 本文回答：导数/梯度到底表示什么？为什么"沿负梯度方向"就能让损失下降？链式法则如何支撑反向传播？

## 一、直觉与通俗解读

- **导数**就是"**瞬时变化率**"，即函数曲线在某点的**斜率**——它告诉你"再往前走一小步，函数值会怎么变"。
- **梯度**是导数在多维的推广，是一个**向量**，指向函数**上升最快**的方向。所以它的反方向就是"下降最快"的方向。
- **链式法则**是"**逐层传递变化**"的规则：如果 A 影响 B、B 影响 C，那么 A 对 C 的影响 = (A 对 B 的影响) × (B 对 C 的影响)。深度网络的反向传播就是链式法则的层层套用。

## 二、严谨定义与原理

### 2.1 导数与偏导数

单变量导数：

$$
f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}
$$

多变量函数 $f(x_1,\dots,x_n)$ 对某一变量的**偏导数** $\dfrac{\partial f}{\partial x_i}$，是把其它变量视为常数后的导数。

### 2.2 梯度

梯度是所有偏导数组成的向量：

$$
\nabla f = \left(\frac{\partial f}{\partial x_1}, \dots, \frac{\partial f}{\partial x_n}\right)^\top
$$

**核心性质**：$\nabla f$ 指向 $f$ 增长最快的方向，$-\nabla f$ 指向下降最快的方向。这就是梯度下降 $\theta \leftarrow \theta - \eta\nabla f$ 的数学依据。

### 2.3 链式法则

若 $y=f(u)$ 且 $u=g(x)$，则

$$
\frac{dy}{dx} = \frac{dy}{du}\cdot\frac{du}{dx}
$$

多变量情形推广为雅可比矩阵（Jacobian）相乘。**反向传播 = 从输出到输入反复应用链式法则，一次回传得到所有参数的梯度。**

### 2.4 二阶信息（了解）

二阶导（多维为 Hessian 矩阵）刻画曲率：曲率决定了在最优点附近函数是"陡峭的碗"还是"平缓的盘"，影响优化的难易与学习率选择。

## 三、案例解析：用梯度找 $f(x)=x^2$ 的最小值

目标：最小化 $f(x)=x^2$，其导数 $f'(x)=2x$。取初值 $x_0=4$，学习率 $\eta=0.1$，迭代 $x \leftarrow x - \eta\cdot 2x = x(1-0.2)=0.8x$：

| 步 | $x$ | $f(x)=x^2$ |
| --- | --- | --- |
| 0 | 4.000 | 16.000 |
| 1 | 3.200 | 10.240 |
| 2 | 2.560 | 6.554 |
| 3 | 2.048 | 4.194 |
| ... | ... | ... |
| 10 | 0.429 | 0.184 |

$x$ 稳步趋向最优点 $0$，$f(x)$ 单调下降。这就是"沿负梯度方向迭代"的完整闭环，也是训练神经网络的最小化过程的一维缩影。

## 四、常见误区与边界

- **误区："梯度为 0 就是最小值"**：梯度为 0 只说明是**驻点**，可能是极小、极大或鞍点；深度学习的高维损失面充满鞍点。
- **误区："学习率越大收敛越快"**：过大会越过最优点甚至发散（如上例若 $\eta=1.1$ 则 $x$ 会放大）。
- **不可导点**：如 ReLU 在 0 处不可导，工程上用次梯度（取 0）处理。

## 五、小结与延伸阅读

- 导数是斜率、梯度是"最快上升方向"，负梯度用于最小化。
- 链式法则把复合函数的梯度拆成逐层相乘，是反向传播的数学基础。
