# 001 · 扩散模型 Diffusion 基础

> 本文用大白话回答：Stable Diffusion、DALL·E 背后那套"加噪再去噪"到底在干什么？为什么从随机噪声也能"画"出图？和 GAN 比有什么不同？
>
> 读完你会知道：扩散模型 = 先学会"怎么一步步去噪"，推理时从纯噪声出发，反复去噪就得到一张新图。

## 一、一句话先说清

**扩散模型（Diffusion Model）= 训练时把清晰图像逐步加噪变糊，再教神经网络学会"倒着走"——从噪声一步步还原清晰图；生成时从纯噪声出发，重复去噪得到新样本。**

两个过程：

- **前向扩散（加噪）**：$x_0 \to x_1 \to \dots \to x_T$，图像越来越像随机噪声（固定公式，不需学习）；
- **反向去噪（生成）**：$x_T \to \dots \to x_0$，用神经网络一步步预测并去掉噪声（需要学习）。

## 二、打个比方：照片放进搅拌机，再学会"倒放"

想象一张清晰照片：

1. **加噪阶段**：每天往照片里撒一点"雪花"，365 天后完全看不出原图——这就是前向扩散，把数据变成纯噪声。
2. **学去噪**：给模型看"半噪声 vs 更噪一点"的配对，让它学会"这一小步该怎么去噪"。
3. **生成阶段**：从一张**全是雪花**的图出发，让模型反复执行"去一点噪"，50~100 步后突然变清晰——得到一张**训练集里没出现过**的新图。

| 生活类比 | 扩散模型对应 |
| --- | --- |
| 照片变糊 | 前向加高斯噪声 |
| 学会"倒放录像" | 训练去噪网络 |
| 从雪花屏变出画 | 从 $x_T$ 采样生成 $x_0$ |

## 三、它到底解决什么问题

### 问题 1：GAN 训练不稳定，扩散更"稳"

**GAN（生成对抗网络）** 让生成器与判别器"左右互搏"，常出现模式崩塌、训练难收敛。

| 维度 | GAN | 扩散模型 |
| --- | --- | --- |
| 训练目标 | 对抗博弈 | 去噪回归（MSE 等），更稳定 |
| 生成质量 | 高但易崩 | 高且稳定，细节好 |
| 推理速度 | 一次前向，快 | 需多步去噪，慢 |
| 代表应用 | 早期人脸生成 | Stable Diffusion、DALL·E 2/3、Imagen |

扩散用"小步去噪"换"训练稳定 + 质量高"；工程上用少步蒸馏、Latent Diffusion 加速（见下文）。

### 问题 2：前向过程怎么加噪——不需要学习

设原图为 $x_0$，每步加少量高斯噪声：

$$
q(x_t \mid x_{t-1}) = \mathcal{N}\big(x_t;\ \sqrt{1-\beta_t}\, x_{t-1},\ \beta_t \mathbf{I}\big)
$$

$\beta_t$ 是噪声日程（noise schedule）。重要性质：可直接从 $x_0$ 跳到任意 $t$：

$$
q(x_t \mid x_0) = \mathcal{N}\big(x_t;\ \sqrt{\bar{\alpha}_t}\, x_0,\ (1-\bar{\alpha}_t)\mathbf{I}\big)
$$

其中 $\alpha_t = 1-\beta_t$，$\bar{\alpha}_t = \prod_{s=1}^{t}\alpha_s$。

> 对齐：$\bar{\alpha}_t$ 越大，$x_t$ 越保留原图；越小越接近纯噪声。前向就是"按日程把图磨成噪声粉"。

### 问题 3：反向怎么去噪——网络预测什么

反向学习 $p_\theta(x_{t-1}\mid x_t)$。DDPM 经典做法：让网络 $\epsilon_\theta(x_t, t)$ **预测加到 $x_0$ 上的噪声** $\epsilon$，损失为：

$$
L = \mathbb{E}_{x_0,\epsilon,t}\big[\|\epsilon - \epsilon_\theta(x_t, t)\|^2\big]
$$

其中 $x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1-\bar{\alpha}_t}\,\epsilon$，$\epsilon \sim \mathcal{N}(0,\mathbf{I})$，$t$ 随机采样。

> 对齐：训练就是"给一张半噪图 + 时间步 $t$，猜当初撒了多少噪"；猜得越准，倒放去噪越靠谱。

推理时从 $x_T \sim \mathcal{N}(0,\mathbf{I})$ 出发，按学到的规则迭代 $T$ 步（或更少步 + 快速采样器）得到 $x_0$。

### 问题 4：算太慢怎么办——Latent Diffusion（Stable Diffusion）

在像素空间做扩散，分辨率高时极慢。**Latent Diffusion** 先用 VAE 把图像压到低维**潜空间（latent）**，在 latent 里做扩散，最后再解码回像素——Stable Diffusion 即此路线，并加入**文本条件**（cross-attention）实现文生图。

## 四、专业视角（与大白话对齐）

| 概念 | 含义 |
| --- | --- |
| DDPM | Denoising Diffusion Probabilistic Models，去噪扩散概率模型 |
| 噪声日程 $\beta_t$ | 控制每步加噪多少；常用 linear / cosine schedule |
| 条件生成 | 以文本、类别、边缘图等为条件 $c$，学 $\epsilon_\theta(x_t, t, c)$ |
| Classifier-Free Guidance | 训练时随机丢条件，推理时放大条件影响，提高 prompt 遵循度 |
| 采样器 | DDIM、DPM-Solver 等，用更少步数近似去噪 |

与 [06 计算机视觉](../06-计算机视觉/000-分类总览与知识图谱.md) 的关系：CNN/ViT 常作去噪网络的 backbone；生成是 CV 的"造数据"方向，与 [004 检测分割](../06-计算机视觉/004-目标检测与图像分割.md) 等判别任务互补。

## 五、案例解析：一维玩具扩散的直觉

设一维"图像"只是数轴上的一个点 $x_0=1.0$。前向 3 步后：

| 步数 $t$ | $\bar{\alpha}_t$（示意） | $x_t$ 大致形态 |
| --- | --- | --- |
| 0 | 1.0 | 清晰点 1.0 |
| 1 | 0.9 | 1.0 + 小噪声 |
| 2 | 0.5 | 噪声与信号各半 |
| 3 | 0.1 | 几乎纯噪声 |

训练时随机抽 $t=2$，给模型看 $x_t$，让它预测混入的 $\epsilon$；推理时从 $x_3$（纯噪）出发，用学到的去噪规则一步步回到 $x_0$ 附近——得到**新**样本（可能 0.98、1.02 等，服从学出的分布）。

高维图像同理：每个像素/每个 latent 维度都在"加噪-去噪"框架下运作。

## 六、常见误区与边界

- **误区："扩散就是 GAN 换皮"**：目标函数与采样机制完全不同；扩散是显式多步马尔可夫去噪。
- **误区："步数越多越好"**：超过一定步数收益递减；工程上用快速采样器在 20~50 步达到可接受质量。
- **误区："生成图一定真实可信"**：会出错细节（手指、文字），需人工或检测模型筛；涉及 deepfake 伦理与合规。
- **边界**：视频/3D 生成在扩散框架上扩展（时序一致性、多视角），算力与数据需求更高。

## 七、一句话总结

- 扩散模型 = 前向加噪（固定）+ 反向去噪（学习）；生成从噪声迭代还原。
- 比 GAN 训练稳、质量高，但推理慢；Latent Diffusion + 条件控制是 Stable Diffusion 类产品的工程基础。
