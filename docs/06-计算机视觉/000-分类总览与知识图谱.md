## 二、通俗理解本分类

一张图片在计算机眼里就是一个**数字网格**（每个像素是几个数）。计算机视觉要做的是从这堆数字里读出含义：

- **卷积**像用一个"**小滤镜**"在图上滑动，专门检测某种局部花纹（边缘、纹理、部件）；
- 多层卷积**层层抽象**：底层看边缘，中层看部件，高层看整体物体；
- 不同任务问不同问题：**分类**问"这是什么"，**检测**问"有什么、在哪"，**分割**问"每个像素属于谁"；
- **ViT** 则把图切成小块，用自注意力"通读全局"，是大规模预训练时代的重要路线。

## 四、学习建议

1. 先理解卷积运算与感受野，这是 CV 的第一性原理。
2. 沿经典架构的演进理解"为什么要更深、残差如何让深层可训练"。
3. 检测与分割是应用主力，理解其评价指标（IoU/mAP）很重要。
4. 学完 CNN 路线后读 [005 ViT](./005-视觉Transformer ViT.md)，对比"卷积局部先验"与"Transformer 全局注意力"。
5. 本分类依赖 [03-深度学习基础](../03-深度学习基础/000-分类总览与知识图谱.md)；ViT 与 [05 大语言模型](../05-大语言模型与Transformer/000-分类总览与知识图谱.md) 紧密相关。

## 五、小结

- CV = 把图像张量经卷积或 patch+Transformer 抽象，再服务于分类/检测/分割等任务。
- 卷积的"局部连接 + 权值共享"是 CNN 的关键归纳偏置；ViT 用大数据预训练弥补弱先验。
- ResNet 让深层 CNN 可训；ViT 与 Swin 等拓展了视觉 backbone 的选择空间。
