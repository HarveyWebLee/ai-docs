# 002 · 卷积神经网络 CNN

> 本文回答：CNN 由哪些层组成？池化在做什么？为什么"卷积—池化"堆叠能实现从局部到整体的层级抽象？

## 一、直觉与通俗解读

CNN 是把 [卷积](./001-图像表示与卷积运算.md) 组装成的"**图像理解流水线**"：

- **卷积层**：提取局部特征（边缘 → 部件 → 物体）。
- **池化层**：把特征图"**缩小、抓重点**"，保留最显著的信息、丢弃冗余，既降计算量又带来一定的位置鲁棒性。
- **全连接层**：在流水线末端把提取到的高级特征汇总，做出最终判断（是猫还是狗）。

整体就像"**先层层看细节、再逐步概括、最后下结论**"。

## 二、严谨定义与原理

### 2.1 典型 CNN 的组成

$$
\text{输入} \to [\text{卷积} \to \text{激活(ReLU)} \to \text{池化}]\times n \to \text{展平} \to \text{全连接} \to \text{softmax}
$$

- **卷积 + ReLU**：提取特征并引入非线性（见 [03/001](../03-深度学习基础/001-神经网络结构.md)）。
- **池化（Pooling）**：最常见是**最大池化**，取局部窗口最大值；也有平均池化。
- **全连接 + softmax**：输出各类别概率。

### 2.2 层级特征抽象

随着层数加深，感受野逐渐变大，特征也从具体到抽象：

| 层级 | 学到的特征 |
| --- | --- |
| 浅层 | 边缘、颜色、方向 |
| 中层 | 纹理、局部部件（眼睛、车轮） |
| 深层 | 完整物体、语义概念 |

这种"局部 → 整体"的层级组合，正是 CNN 强大的根源。

### 2.3 通道与参数

每个卷积层有多个卷积核，每个核输出一个特征通道。参数量 = $k\times k\times C_{in}\times C_{out}$，远少于同规模全连接层——这是权值共享的红利。

## 三、案例解析：MNIST 手写数字识别的信息流

以经典的手写数字（28×28 灰度图）分类为例，追踪张量形状的变化：

1. 输入 `28×28×1`。
2. 卷积（多个 3×3 核）+ ReLU → 得到多通道特征图，如 `28×28×32`（提取边缘/笔画）。
3. 最大池化 `2×2` → `14×14×32`（尺寸减半，保留显著响应）。
4. 再一轮卷积 + 池化 → `7×7×64`（更抽象的部件特征）。
5. 展平为向量 → 全连接 → softmax → 输出 `10` 个类别的概率。

模型据此把图片分类为 0~9 中的某个数字。**从原始像素到类别概率，CNN 通过卷积-池化的反复堆叠完成了层级抽象**，这也是它在图像任务上远超全连接网络的原因。

## 四、常见误区与边界

- **误区："池化必不可少"**：现代架构有时用带步长的卷积替代池化实现下采样。
- **误区："CNN 有旋转不变性"**：CNN 主要具备**平移**不变性（+池化的局部鲁棒性），对大幅旋转/缩放并不天然不变，常靠数据增强弥补。
- **边界**：CNN 感受野随深度增长但仍偏局部，全局长程关系建模弱于 Transformer。

## 五、小结与延伸阅读

- CNN = 卷积（提特征）+ 池化（抓重点降维）+ 全连接（做判断）。
- 卷积-池化堆叠实现从边缘到物体的层级抽象，参数远少于全连接。
